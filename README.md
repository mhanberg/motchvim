# motchvim

Welcome to `motchvim`.

My Neovim config, packaged as a lazy.nvim plugin. 

```lua
require("lazy").setup {
  spec = {
    { dir = "mhanberg/motchvim", import = "motchvim.plugins" },
  },
}
```
